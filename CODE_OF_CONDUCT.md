# Code of Conduct

RiCo welcomes constructive technical discussion.

Contributors are expected to:

- Focus on architecture and evidence.
- Respect differing technical viewpoints.
- Distinguish observation from authority.
- Support claims with reasoning.
- Preserve professional conduct.

RiCo exists to improve trustworthy runtime governance through collaborative architectural discussion.

---

Humans First.

Continuity Always.

Architecture Must Survive.
