# Security Policy

## Purpose

RiCo is an architectural governance project. As the repository evolves to include reference implementations and runtime software, security will become an increasingly important part of the project.

Until then, this repository primarily contains architectural documentation and reference materials.

---

## Reporting Security Issues

If you believe you have identified a security issue related to RiCo or associated software, please report it privately.

Please do not disclose potential vulnerabilities publicly before they have been reviewed.

Report security concerns to:

**Email:** richard.colimon@manchine.ai

---

## Scope

This policy currently applies to:

- Reference implementations
- Runtime governance software
- Repository infrastructure
- Future RiCo software releases

It does not apply to third-party projects that reference or discuss RiCo.

---

## Responsible Disclosure

We appreciate responsible disclosure and will acknowledge legitimate reports after review.

As the project matures, this policy may expand to include version support, coordinated disclosure timelines, and security advisories.
