# RiCo Specification

## Purpose

The Specification directory contains the normative architectural definitions for Runtime Integrity Control (RiCo).

Unlike explanatory documentation, the files in this directory define the stable architectural behavior expected across implementations.

## Current Specifications

- Runtime Execution Boundary (REB)

## Planned Specifications

- Observation Specification
- Governance Receipt Specification
- Runtime State Model
- Execution Contract
- Interface Definitions

## Design Principle

Documentation explains architecture.

Specifications define architecture.
